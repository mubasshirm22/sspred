Attachable companion analyses
These pull focused outputs from the lab’s other tools directly into the ProtPipe report. Keep them off for the lightest workflow, or enable them when you want disorder and secondary-structure context in one place.
Advanced / experimental modules
Useful when you want broader coverage, but slower or more brittle than the core path.
All annotation tools run in parallel with a shared 10-minute timeout. BLAST runs concurrently — if BLAST completes later, its results are added at the end. If any single tool fails or times out, the rest continue and results are shown.

Accepted Inputs
  • Raw amino acid string (single-letter codes)
  • Single FASTA record (with or without header)
  • UniProt accession (e.g. P04637)
  • NCBI protein accession (e.g. NP_000535.3, XP_, WP_)
  • NCBI mRNA / transcript (e.g. NM_005228.5, XM_) → auto-resolved
  • WGS contig (e.g. GAJC01020720.1) → auto-resolved
  • NCBI gene ID (numeric) → auto-resolved
  • Uploaded .fasta file
What You Get
  • Sequence properties (MW, pI, GRAVY, stability)
  • Top 10 BLAST hits vs SwissProt
  • Pfam domain hits with coordinates
  • Signal peptide detection and cleavage site
  • Transmembrane helix count and topology
  • Optional DisorderPred and SSPred companion panels
  • SVG domain architecture figure
  • Downloadable JSON / TXT / SVG report
Typical Runtime
  • Sequence retrieval: <5 seconds
  • Sequence properties: instant
  • BLAST: 2–10 minutes (SwissProt)
  • Pfam HMMER: ~30–60 seconds
  • Phobius: ~30 seconds
  • BLAST, HMMER, and Phobius run in parallel
